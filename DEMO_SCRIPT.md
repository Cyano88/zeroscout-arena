# Demo Script

## 60-Second Version

1. "ZeroScout turns a builder's repo and demo into a verified public project profile backed by 0G."
2. Open the Create page.
3. Submit a project with repo, demo, round, and the exact 0G usage claim.
4. "The AI analyst creates a project brief, next steps, risk list, and share kit."
5. Show the verification panel.
6. "The canonical profile JSON is stored on 0G Storage mainnet. The app indexes only lightweight metadata."
7. Open the public project profile.
8. Show storage root, content hash, AI provider, project strength, fixes, and share kit.
9. Create or open a second profile for the same project.
10. Show progress since last proof: what improved and what should ship next.
11. Open Compare and compare two verified project pages.
12. "For Zero Cup, this helps teams keep proof and progress visible between cuts. Outside Zero Cup, it works for grants, demo days, accelerators, and ecosystem discovery."

## Proof Line

"If 0G is removed, ZeroScout loses its canonical public memory layer. The verified profile, comparison report, root hash, and verifiable storage artifact are the product surface."
